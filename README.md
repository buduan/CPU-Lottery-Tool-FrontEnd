# 宁诺CPU抽奖工具前端
这是一个开发的比较仓促的项目，实际上到使用的前2天才开工，属于是边用边改。
为了省钱，不买抽奖机，用iPad代替的小工具，也大大的降低我们按照学校要求需要登记记录奖品领取对象以及核验是否填写的时间。
后端具有记录功能，可以快速导出抽奖记录。前端还在完善，因为明年还能用上~

## 功能介绍
- 支持PWA应用，可配合iPadOS引导式访问使用
- 具备前端内置为线下触控环境优化的屏幕键盘
- 自动提交
- 未包含管理后台前端（开发中）
